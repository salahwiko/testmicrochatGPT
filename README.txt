# Project Name
ChatGPT Microservices

# Project Description
This project is a microservices implementation that connects to OpenAI's GPT-3 model to provide a chatbot service and save the question/answer in CSV file
the author is a newbie in the back-end dev and the implementation of microservices and API

# Technologies Used
- Python
- Flask
- OpenAI API

# Setup
1. Clone the repository
2. Navigate to the project directory
3. Install the required dependencies using the command `pip install -r requirements.txt`
4. Run the app using the command `python app.py`

# Installation Guide
